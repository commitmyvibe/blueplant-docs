Hier finden Sie die wichtigsten Begriffe, die Ihnen in blueplant begegnen – kurz und verständlich erklärt.

## Die wichtigsten Begriffe

| Begriff | Bedeutung |
| --- | --- |
| **Kunde** | Das Unternehmen oder die Person, die Sie betreuen. |
| **Standort** | Ein konkreter Ort eines Kunden, an dem Pflanzen stehen. |
| **Pflanze** | Eine Pflanzenart im Katalog, mit ihren Pflegeeigenschaften. |
| **Produkt** | Eine konkrete Pflanze an einem Standort – inkl. Gefäß und Standort-Details. |
| **Termin** | Ein Pflegebesuch an einem Standort. |
| **Serie** | Eine automatisch erzeugte Folge von Terminen aus einem Intervall. |
| **Pflegeergebnis** | Die Dokumentation, was bei einem Termin an einer Pflanze gemacht wurde. |
| **Leistungsnachweis** | Ein Beleg über die geleistete Pflege (keine Rechnung). |
| **fastCare** | Schnelles Erfassen mehrerer Pflanzen auf einmal. |
